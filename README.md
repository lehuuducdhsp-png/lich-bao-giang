# Công cụ Lập Lịch Báo giảng

Website độc lập dùng để đọc file TKB Excel, chọn giáo viên và xuất Lịch Báo giảng.

- Không kết nối với website quản lý lớp học.
- Không dùng Supabase hoặc dữ liệu học sinh.
- File Excel được xử lý trực tiếp trong trình duyệt.
- Cho phép lưu nhiều phiên bản TKB trên thiết bị.
- Phần giáo án luôn để đúng mẫu `(GA  )`.

## GitHub Pages

Vào **Settings → Pages**, chọn **Deploy from a branch**, nhánh **main**, thư mục **/(root)** rồi lưu.
